# Hospital-Management-System-Python-
Developed a full hospital management system in Python using object-oriented programming and file handling. The program manages doctors and patients, allowing users to add, edit, search, and display records through an interactive menu system.

Key features:
• Object-Oriented Design — Built Doctor, Patient, and Manager classes using encapsulation and clean architecture.
• Record Management — Implemented adding, editing, and searching doctor and patient information using structured methods.
• File Integration — Reads and updates doctor and patient records stored in text files, keeping information organized and accessible.
• Interactive CLI Menu — Designed user-friendly menus for navigating doctor and patient options.
• Team Collaboration — Integrated multiple contributors’ work into one cohesive project.
